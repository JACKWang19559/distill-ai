## 1. 注册与登录

### 1.1 邮箱注册

1. 访问应用首页，点击「注册」
2. 填写邮箱、用户名、密码（至少 8 位）
3. 点击「创建账户」即可直接登录，无需邮箱验证

### 1.2 GitHub 登录（可选）

1. 在登录页点击「使用 GitHub 登录」
2. 跳转 GitHub 授权页，授权后自动返回应用并登录

> 首次使用任一方式登录都会自动创建账户。

## 2. 配置 AI 供应商

蒸馏前必须配置至少一个 AI 供应商的 API Key。

### 2.1 添加 API Key

1. 登录后进入「设置 → API 配置」
2. 选择供应商（OpenAI / Anthropic / 通义千问 / DeepSeek / 智谱 GLM）
3. 粘贴 API Key
4. 点击「保存」

API Key 使用 AES-256 加密存储，仅用于调用对应供应商的 API。

### 2.2 切换默认供应商

在 API 配置列表中，点击某条记录的「设为默认」按钮，后续蒸馏任务将使用该供应商。

### 2.3 获取 API Key

| 供应商 | 申请地址 |
|--------|---------|
| OpenAI | https://platform.openai.com/api-keys |
| Anthropic | https://console.anthropic.com/ |
| 通义千问 | https://dashscope.console.aliyun.com/ |
| DeepSeek | https://platform.deepseek.com/ |
| 智谱 GLM | https://open.bigmodel.cn/ |

## 3. 安装浏览器插件

### 3.1 构建插件

```bash
pnpm --filter @distill/extension build
```

构建产物位于 `apps/extension/dist/`。

### 3.2 加载到浏览器

**Chrome**：
1. 打开 `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `apps/extension/dist` 目录

**Edge**：
1. 打开 `edge://extensions`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择 `apps/extension/dist` 目录

加载成功后，浏览器工具栏会出现 Distill 图标。

### 3.3 插件登录

1. 在任意网页点击工具栏的 Distill 图标，打开侧边栏
2. 输入 Web 端注册的邮箱和密码
3. 点击「登录」

> 插件登录态独立于 Web 端，需单独登录。

## 4. 蒸馏网页内容

### 4.1 一键蒸馏

1. 打开任意网页（如公众号文章、博客、新闻）
2. 点击浏览器工具栏的 Distill 图标，打开侧边栏
3. 点击「一键蒸馏」按钮
4. 侧边栏实时显示处理进度
5. 完成后展示：摘要、关键要点、推荐标签
6. 点击「保存到知识库」

### 4.2 蒸馏选中内容

1. 在网页中选中一段文字
2. 右键 → 「Distill: 蒸馏选中内容」
3. 侧边栏仅对选中部分进行蒸馏

## 5. 蒸馏 PDF / 文本 / 视频

在 Web 端「工作台」页面，支持多种输入源：

### 5.1 PDF 文件

1. 切换到「PDF」标签
2. 拖拽或点击上传 PDF 文件（单文件 ≤ 20MB）
3. 点击「开始蒸馏」
4. 系统调用 opendataloader-pdf 提取文本（支持表格、OCR、公式）
5. AI 生成结构化总结

### 5.2 纯文本

1. 切换到「文本」标签
2. 粘贴任意文本或 Markdown
3. 点击「开始蒸馏」

### 5.3 抖音视频

1. 切换到「抖音」标签
2. 粘贴抖音视频分享链接（如 `https://v.douyin.com/xxxxx/`）
3. 点击「开始蒸馏」
4. 系统自动：下载视频 → 分离音频 → ASR 识别文案 → AI 蒸馏

### 5.4 小红书笔记

1. 切换到「小红书」标签
2. 粘贴小红书笔记链接
3. 点击「开始蒸馏」
4. 系统自动：提取图文或视频 → ASR（视频）→ AI 蒸馏

> 视频类蒸馏耗时较长（通常 30 秒 - 2 分钟），取决于视频时长与 ASR 服务。

## 6. 管理知识库

### 6.1 查看知识列表

进入「知识库」页面：
- **卡片视图**：以卡片形式展示，显示标题、摘要、标签、来源、时间
- **列表视图**：紧凑列表，适合快速浏览
- 点击右上角图标切换视图

### 6.2 标签筛选

- 左侧标签栏显示所有标签及数量
- 点击标签筛选对应知识
- 再次点击取消筛选

### 6.3 查看与编辑详情

1. 点击任意知识卡片进入详情页
2. 详情页展示：标题、摘要、关键要点、层级大纲、标签、原文
3. 点击「编辑」可修改标题、标签、补充个人笔记
4. 点击「删除」会弹出确认对话框（删除后不可恢复）

### 6.4 创建标签

1. 进入「设置 → 标签管理」或在知识详情页
2. 点击「新建标签」
3. 输入标签名称、选择颜色
4. 保存后可在蒸馏时或编辑时使用

## 7. 搜索知识

### 7.1 全文搜索

1. 进入「搜索」页面
2. 在搜索框输入关键词
3. 系统在标题、摘要、原文中匹配
4. 结果列表高亮显示匹配关键词

### 7.2 高级筛选

- **来源类型**：网页 / PDF / 文本 / 抖音 / 小红书
- **时间范围**：最近 7 天 / 30 天 / 自定义
- **标签**：多选筛选

### 7.3 搜索技巧

- 多个关键词用空格分隔，表示 AND 关系
- 支持中文分词
- 搜索结果按相关度排序

## 8. 查看知识图谱

### 8.1 打开图谱

进入「图谱」页面，系统自动加载所有知识节点及其关联。

### 8.2 图谱交互

| 操作 | 效果 |
|------|------|
| 拖拽节点 | 移动节点位置 |
| 滚轮缩放 | 放大/缩小图谱 |
| 拖拽空白处 | 平移整个图谱 |
| 点击节点 | 高亮该节点及其关联 |
| 双击节点 | 跳转到知识详情页 |
| 点击边 | 查看关联类型与强度 |

### 8.3 图谱控制

右上角控制面板：
- **布局重置**：恢复初始力导向布局
- **节点筛选**：按标签显示/隐藏节点
- **边阈值**：仅显示关联强度高于阈值的边
- **全屏**：进入全屏模式

### 8.4 关联类型

| 类型 | 说明 |
|------|------|
| related | 共享实体或标签 |
| similar | 内容相似度高 |
| references | 引用关系 |
| extends | 扩展关系 |

## 9. 常见问题

### Q1：蒸馏失败怎么办？

**可能原因与解决**：
1. **API Key 无效**：检查「设置 → API 配置」中的 Key 是否正确
2. **API 余额不足**：登录对应供应商后台查看余额
3. **网络问题**：确保服务器能访问 AI 供应商 API
4. **内容过长**：PDF 或文本超过模型上下文限制，尝试分段

### Q2：插件登录失败？

1. 确认 Web 端已注册账号且能正常登录
2. 检查插件登录地址是否指向正确的 Web 服务
3. 登录接口限流为 5 次/分钟，频繁失败请稍后再试

### Q3：抖音/小红书蒸馏失败？

1. 确认媒体处理服务（端口 8001）已启动
2. 确认服务器已安装 ffmpeg
3. 部分私密笔记或已删除视频无法提取
4. yt-dlp 版本过旧可能导致下载失败，更新到最新版

### Q4：PDF 提取乱码？

1. 扫描件 PDF 需要 OCR 支持，确保安装 Java 11+
2. 使用 `opendataloader-pdf[hybrid]` 模式处理复杂表格
3. 加密 PDF 需先解除密码

### Q5：知识图谱节点太多看不清？

1. 使用「边阈值」过滤弱关联
2. 使用「节点筛选」按标签过滤
3. 滚轮缩放放大局部区域
4. 拖拽节点手动调整布局

### Q6：如何导出知识数据？

目前 MVP 版本暂不支持一键导出，可通过 Prisma Studio 直接访问数据库导出：
```bash
pnpm --filter web db:studio
```

## 💡 使用建议

1. **蒸馏后立即补充笔记**：AI 生成的摘要是起点，加入个人思考才能真正内化
2. **善用标签**：建立自己的标签体系（如 `AI/LLM`、`行业/教育`），便于检索与图谱关联
3. **定期查看图谱**：发现知识点之间的意外关联，激发新想法
4. **选中蒸馏**：长文章可只蒸馏核心段落，避免噪音
5. **多供应商对比**：不同 AI 对同一内容的总结风格不同，可切换供应商对比效果
